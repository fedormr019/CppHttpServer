# CppHttpServer

Легковесный многопоточный HTTP-сервер на C++17 с использованием WinSock.

## Возможности

- Многопоточная обработка запросов с помощью `std::thread`
- Собственный парсер HTTP-запросов
- Собственный роутер для маршрутизации запросов
- Потокобезопасная система логирования
- Логирование в консоль и файл
- Поддержка HTTP-статусов (200, 404)
- Система сборки CMake

## Эндпоинты

| Метод | Маршрут    | Описание                 |
|-------|------------|--------------------------|
| GET   | `/`        | Домашняя страница        |
| GET   | `/ping`    | Проверка работоспособности |
| GET   | `/about`   | Информация о сервере     |

## Используемые технологии

- C++17
- WinSock2
- `std::thread`
- `std::mutex`
- CMake
