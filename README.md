CppHttpServer

Небольшой HTTP-сервер, написанный на C++17 в образовательных целях.

В проекте реализованы:

обработка HTTP GET-запросов;
собственный роутер для маршрутизации запросов;
многопоточная обработка подключений через std::thread;
потокобезопасная система логирования;
запись логов в консоль и файл;
сборка проекта через CMake;
работа с сокетами через WinSock.

Проект создан для изучения сетевого программирования, многопоточности и базовых принципов backend-разработки на C++.
